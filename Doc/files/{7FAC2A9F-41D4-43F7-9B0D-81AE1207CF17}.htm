Pré-requisitos

As exigências mínimas de software e hardware para que seu sistema execute corretamente o X-Maker são:

  • Microsoft Windows (98 ou versão posterior).
  • Delphi 5, 6, 7 ou 2005 na versão Professional ou Enterprise instalado na máquina.
  • Microcomputador com processador Pentium (ou superior), 128 megabytes de memória, 150 megabytes de espaço livre em disco.
