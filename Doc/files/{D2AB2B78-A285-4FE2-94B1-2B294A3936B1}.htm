Instalando

A instalação do X-Maker em seu computador é feita pelo programa Instalar, através do arquivo disponível em nosso site para download.

Para instalar o X-Maker através do arquivo de download:

  1. Faça o download do arquivo em nosso site http://www.xmaker.com.br na seção de Downloads.
  2. Execute o arquivo em seu computador.
  3. Siga as instruções mostradas na tela.

Pasta de Instalação:

A pasta de Instalação do X-Maker padrão é – C:\xMaker4 - sugerimos que esta pasta seja mantida. Esta recomendação está relacionada a uma limitação do sistema operacional relativa ao comprimento máximo de linhas de comando executado via MS-DOS, pois a compilação do projeto pelo X-Maker se faz através de linhas de comandos do MS-DOS. Recomenda-se que a nova pasta esteja localizada imediatamente abaixo da pasta raiz escolhida, Exemplos: C:\XMaker4 - D:\XMaker4 – F:\XMaker4 - …

FireBird:

O FireBird é o gerenciador de banco dados utilizado pelo X-Maker. Sua instalação é necessária para o funcionamento correto do X-Maker. Caso o InterBase 6.x ( ou superior ) esteja instalado em seu computador a instalação do FireBird não será necessária, por ser compatível com o FireBird. Qualquer versão do Interbase inferior a 6.x deverá ser desinstalada antes da instalação do FireBird.
