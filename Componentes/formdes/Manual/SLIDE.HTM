TSlideTransition

Unit

teSlide

Declaration

TSlideTransition = class(TTimedTransitionEffect)

Description

The destination image moves progressively into the scene.

Properties

Direction

Declaration: property Direction: Word read FDirection write SetDirection default tedDown;
Description: The allowed directions are tedRight, tedLeft, tedDown and tedUp.
See also: Direction constants

ElasticDst

Declaration: ElasticDst: Boolean;
Description: If False (the default), the destination image is moved into the scene. If True, then the destination image is progressively unfolded.

ElasticSrc

Declaration: ElasticSrc: Boolean;
Description: If False (the default), the source image is completely static. If True, then the source image is progressively shrunk.
