TTimedTransitionEffect

Unit

teTimed

Declaration

TTimedTransitionEffect = class(TTransitionEffect)

Description

This is the base class for all the timed transitions. A timed transition will last Milliseconds msecs. You’ll never have to instantiate TTimedTransitionEffect, because by itself it is inoperative.

Methods

Methods

ExecuteFrame

Declaration: procedure ExecuteFrame(Data: TTETransitionData; CurrentFrame, Step: Word); virtual; abstract;
Description: This abstract method has to be overriden in all derived classes to execute the transition. CurrentFrame is the frame number (from 1 to the maximum number of frames). Step is the difference between the last executed frame and CurrentFrame. On slow machines the number of frames executed could be less than on fast machines, if Milliseconds is not great enough. This way, the timed transitions are not CPU dependent. The best results are obtained when the time it takes to complete for each frame is relatively stable.

Finalize

Declaration: procedure Finalize; virtual;
Description: It is a method that can be overriden in derived classes to execute custom code after the termination of the transition.
See also: Initialize

Initialize

Declaration: procedure Initialize(Data: TTETransitionData; var Frames: Word); virtual; abstract;
Description: It is an abstract method that has to be overriden in all derived classes. Its mission is to give a chance to properly initialize the transition local variables just before it starts. You also have to assign to Frames the maximum number of frames that can be achieved.
See also: Finalize
