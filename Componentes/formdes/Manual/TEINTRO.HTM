Transitions introduction

A transition is a graphical effect that starts with a source image and ends with a destination image. They are widely used in TV and video, and are ideal for multimedia applications.

These transitions are independent of the CPU speed, completely flicker free and the footprint is minimum.

FormContainer is fully integrated with the transitions, but it is possible to use them with any other control.

If we have, for example, a TImage called MyImage and want to change its picture using a TFuseTransition during 0.5 seconds:

	var
	  Fuse: TFuseTransition;
	begin
	  Fuse := TFuseTransition.Create;
	  try
	    Fuse.Milliseconds := 500;
	    Fuse.Prepare(MyImage, MyImage.ClientRect);
	    MyImage.Picture.LoadFromFile('NewPicture.bmp');
	    Fuse.Execute;
	  finally
	    Fuse.Free;
	  end;
	end;

There are several transitions available, and it is possible to create custom transitions.
