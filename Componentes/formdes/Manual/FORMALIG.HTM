TFCFormAlign

Unit

FormCont

Declaration

TFCFormAlign = (fcfaDefault, fcfaNone, fcfaCenter, fcfaClient, fcfaTopLeft);

Description

These are the different possibilities to align a form into a FormContainer:
 
fcfaNone: it doesn’t adjust the size and position of the form.
fcfaCenter: it maintains the size, but positions it at the center of the FormContainer.
fcfaClient: the form will cover the client area of the FormContainer.
fcfaTopLeft: it maintains the size, but positions it at the top left corner of the FormContainer.
fcfaDefault: the alignment will depend on the value of these properties of the form:
 
BorderIcons: if biMaximize is set, then it is equivalent to fcfaClient. If not, the following applies:
 
Position: if its value is poDesigned, then it is equivalent to fcfaNone, else if it is poScreenCenter, it is equivalent to fcfaCenter. Any other value will default to fcfaTopLeft.
