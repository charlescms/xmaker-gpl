TFormTransitions

Unit

teForm

Declaration

TFormTransitions = class(TComponent)

Description

By dropping this component into a form you can achieve a transition effect when you call Show or ShowModal. It is designed to be used with standard forms. If you insert the form into a FormContainer then it will have no effect. The most simple use is to show a form completely flickerfree. In that case, you only have to drop the component into the form. If you prefer a transition effect then you have to assign it to ShowTransition.

Properties

Enabled

Declaration: property Enabled: Boolean read FEnabled write SetEnabled default True;
Description: The component will take no effect if it isn't enabled.

DestroyTransitions

Declaration: property DestroyTransitions: Boolean read FDestroyTransitions write FDestroyTransitions default True;
Description: If True the assigned transition will be freed along with the component.

ShowTransition

Declaration: property ShowTransition: TTransitionEffect read FShowTransition write SetShowTransition default nil;
Description: This is the transition to use when showing the form. If nil (the default) a FlickerFree transition will be used.
