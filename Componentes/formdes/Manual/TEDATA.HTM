TTETransitionData

Unit

TransEff

Declaration

TTETransitionData = class(TObject)

Description

Provides all the information needed to perform the transition.

Properties

Bitmap

Declaration: property Bitmap: TBitmap read FBitmap;
Description: It is the offscreen bitmap. If no offscreen bitmap is needed, then it returns nil.
See also: UseOffScreenBmp

Canvas

Declaration: property Canvas: TCanvas read GetCanvas;
Description: It is the canvas where you have to render the transition. When using an offscreen bitmap, it is the canvas of that bitmap. When the transition is directly rendered to the screen, it is the canvas of the window where it takes place. This way, you can safely use it without caring if you are working with an offscreen bitmap or not.

DstBmp

Declaration: property DstBmp: TBitmap read FDstBmp;
Description: It is the destination bitmap. If no destination bitmap is needed, then it returns nil.
See also: NeedDstImage

Height

Declaration: property Height: Integer read FHeight;
Description: It is the height of the transition surface.
See also: Width

SrcBmp

Declaration: property SrcBmp: TBitmap read FDstBmp;
Description: It is the source bitmap. If no source bitmap is needed, then it returns nil.
See also: NeedSrcImage

Width

Declaration: property Width: Integer read FWidth;
Description: It is the width of the transition surface.
See also: Height
