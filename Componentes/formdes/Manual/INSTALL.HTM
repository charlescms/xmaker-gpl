Installation

By the moment you can install it in Delphi 3 and Delphi 4:

  1. Uninstall any previous version.
  2. Use 'File\Open...' menu item of Delphi IDE to open 'fcte.dpk' file typically located in the '\FormCont\lib\DelphiX' directory (where 'X' is 3 or 4, depending on the Delphi version)
  3. In "Package ..." window click "Compile" and then 'Install'.
  4. The components will be located into the FormContainer palette. If you had installed a previous version of this library Delphi seems to remember where the components were previously located, so it will probably remain in the System palette.
  5. Enjoy it!

Uninstallation

  1. Use 'Component\Install Packages...' menu item of Delphi IDE.
  2. Select 'FormContainer & Transition effects' package and click on the 'Remove' push button.
  3. Launch the Add/Remove Programs applet out of the Control Panel, highlighting 'FormContainer & Transition effects' and clicking on the 'Add/Remove...' push button.
