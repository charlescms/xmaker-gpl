TTEChrono

Unit

teChrono

Declaration

TTEChrono = class(TObject)

Description

This is an utility class used internally by TTimedTransitionEffect to accurately measure the time. It has been made public so you can use it for your own needs.

Properties Methods

Properties

Passes

Declaration: property Passes: Longint read FPasses;
Description: Counts the number of calls to the Start method since the last Reset.
See also: Start Reset

Paused

Declaration: property Paused: Boolean read FPaused;
Description: Indicates if the chronograph is not measuring time at that moment.
See also: Resetted

Resets

Declaration: property Resets: Longint read FResets;
Description: Counts the total number of calls to Reset.
See also: Reset

Resetted

Declaration: property Working: Boolean read FWorking;
Description: Indicates if the chronograph is currently resetted.
See also: Paused

Methods

Milliseconds

Declaration: function Milliseconds: Double;
Description: The elapsed time in milliseconds. Note that its precision can be so high that a double value is returned.

Pause

Declaration: procedure Pause;
Description: Pauses the chrono. If you call Start without calling Reset before, the previously elapsed time is preserved.
See also: Start Reset Milliseconds

Reset

Declaration: procedure Reset;
Description: Resets the chrono so the elapsed time is set to 0.
See also: Start Pause Milliseconds

Start

Declaration: procedure Start;
Description: Starts to measure the time.
See also: Pause Reset Milliseconds
