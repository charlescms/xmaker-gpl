TBlendTransition

Unit

teBlend

Declaration

TBlendTransition = class(TTimedTransitionEffect)

Description

It performs an alpha blending. This kind of effect can't be used in paletted device modes (in example, 16 or 256 colors video mode), so in that case it is emulated applying a FuseTransition with Style set to 1. The best results both in speed and quality are achieved when 24 bits per pixel are used.
