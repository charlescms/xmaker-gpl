TFuseTransition

Unit

teFuse

Declaration

TFuseTransition = class(TTimedTransitionEffect)

Description

Carries out a fuse. It is fast and has several styles.

Properties

CountOfStyles

Declaration: property CountOfStyles: Integer read FCountOfStyles;
Description: Returns the current number of styles available.
See also: Style StyleToUse

Style

Declaration: property Style: Word read FStyle write SetStyle default 1;
Description: This is the proposed style to use, in the range from 0 to CountOfStyles. The 0 value randomly uses a different style for each transition executed, where the style 1 has a 33% probability.
See also: CountOfStyles StyleToUse

Methods

StyleToUse

Declaration: function StyleToUse: Word;
Description: This is the actual style to use, in the range from 1 to CountOfStyles. Its value depends on the values of Style and Reversed
See also: CountOfStyles Style
