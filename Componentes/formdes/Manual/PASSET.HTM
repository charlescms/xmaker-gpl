TTEPassSettingType

Unit

TransEff

Declaration

TTEPassSettingType = (teOnePass, teTwoPasses, tePaletteDependent);

Description

The possible values to set the number of passes to execute in a transition are:
teOnePass: Only one pass will be executed.
teTwoPasses: Two passes if possible.
tePaletteDependent: One pass if the video is not in 256 colors mode, else two passes if possible.
