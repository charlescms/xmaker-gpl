Versions

1.3

  1. Memory and resources leaks have been fixed.
  2. Slide, Push and Drip transitions have been fixed.
  3. Enhanced rendering of controls.
  4. AdjustBmpForTransition and GetSnapShotImage now accept a parameter giving the desired pixel format.
  5. RenderControl has 3 extra parameters.
  6. New utility procedures and functions: ChangePixelFormat, DevicePixelFormat, GetBytesPerScanline and RGBDevice (for internal use in custom transitions).
  7. New ClientCoordinates property for TTransitionEffect.
  8. New GetPixelFormat function (for internal use in custom transitions).
  9. New UnUpdateRect property for TTransitionEffect (for internal use in custom transitions).
  10. New OnStartTransition end OnEndTransition events for TTransitionEffect.
  11. Transition disabling with TEGlobalDisabled global variable and the Enabled and FlickerFreeWhenDisabled properties for TTransitionEffect.
  12. New alpha-blending transition.
  13. New TFormTransitions component.

1.2b

  1. Push and Slide transitions are much faster now.
  2. Some bugs fixed in the Drip and Slide transitions.
  3. Push and Slide default direction has changed to tedDown because it minimizes the vertical retrace effects.
  4. The Fuse transition now has 5 new styles. The 0 value randomly uses a different style for each transition executed, where the style 1 has a 33% probability.
  5. The demo project now uses external bmp files instead of internal resources. This means that it isn't an standalone executable anymore because it needs these bmps.
  6. 'Line too long' error fixed when installing the Delphi 4 version.
  7. The library is now installed as a package. Please follow the new installation steps.
  8. The source code of FormContainer component is included in all registered versions.

1.2

  1. First release version
  2. More units added, due to a great reorganization of the code. You'll possibly have to include 'teRender' in some uses clauses.
  3. Much better palette handling under 256 colors video mode.
  4. Custom transitions are now much more easy to develope.
  5. More correctly rendered controls.
  6. The OnActivate and OnDeactivate events of a child form are now used as OnEnter and OnExit replacements.
  7. The function GetSnapShotImage now creates and returns a TBitmap. If you were using splash screens you should modify your code (see the About.pas of the Demo project).
  8. RegisterTEControl has changed its parameters.
  9. The global function RenderControl is now public.
  10. The class TTEChrono is now public.
  11. BorderStyle property of TFormContainer

Beta 1.1

  1. Delphi 4 compatible
  2. Global FlickerFreeTransition object
  3. More correctly rendered controls (TAnimate and TComboBox, for example)
  4. When a control is wrongly rendered, it is repainted when the transition has finished (ActiveX controls, for example)
  5. Fine-tuning of control rendering using global method RegisterTEControl
  6. Global method PalettedDevice
  7. BackgroundOptions in TFormContainer
  8. ShowFormEx has changed its parameters
  9. Updated documentation
  10. The nag messages of the evaluation version are hidden when the IDE is running

Beta 1.0

  1. First public version
