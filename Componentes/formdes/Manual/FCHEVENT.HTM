TFCFormChangeEvent

Unit

FormCont

Declaration

TFormContChangeEvent = procedure(Sender: TObject; const OldForm, NewForm: TCustomForm; var CanChange: Boolean) of object;

Description

This event is fired each time that ShowForm or ShowFormEx are called. OldForm is the form that was visible until now (nil if no one was visible), and NewForm is the form that is going to be showed (nil if no one is going to be visible). If CanChange is set to False then the operation is aborted.
