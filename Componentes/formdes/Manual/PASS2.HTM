TTEPass2OptionsType

Unit

TransEff

Declaration

TTEPass2OptionsType = record

Description

Configuration of the second pass if it exists.

Fields

DistributedTime

Declaration: DistributedTime: Boolean;
Description: If False (the default), the transition will spend Milliseconds milliseconds on each pass. If True, each pass will spend half of the time.

Reversed

Declaration: Reversed: Boolean;
Description: If False (the default), both passes use the same parameters. If True, the first pass is the reversed of the second, and the second follows the settings of the transition.

UseSolidColor

Declaration: UseSolidColor: Boolean;
Description: If True (the default), the intermediate image is simply a solid color. If False, the intermediate image is obtained at the time Prepare2ndPass is invoked.

SolidColor

Declaration: SolidColor: TColor;
Description: If UseSolidColor is True, this is the color to use. If it is clNone, then it will use the color of the control of reference.
