AdjustBmpForTransition

Unit

teRender

Declaration

AdjustBmpForTransition(Bmp: TBitmap; Palette: HPALETTE; Width, Height: Integer; PixelFormat: TPixelFormat);

Description

This global function sets Bmp to the appropiate format for the transition. If you have to use your own bitmaps in a custom transition, you should call this function to make sure they will work fine (unless it is a mask bitmap).
